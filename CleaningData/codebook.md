# Project Description

The purpose of this project is to combine the "test" and "training" data from the Human Activity Recognition Using Smartphones Dataset. In addition it provides a mean of the columns that contained standard deviation/mean data

# Study design and data processing
## Collection of the raw Data

The raw data originated from the "Human Activity Recognition Using Smartphones Dataset". Details regarding the raw data can be found in the folder "UCI HAR Dataset"
